# 🎨 BRIEF: TRÌNH DIỄN DEMO & CHIẾN LƯỢC KINH DOANH

## 🎯 Mục tiêu Demo (The "Wow" Effect)
Khách hàng (Manager/Sếp) phải cảm thấy hệ thống này "thông minh vượt trội" nhưng "dễ dùng như chat Zalo".

## 🛠️ Yêu cầu Kỹ thuật cho Claude:
1. **Deployment:** Chuẩn bị sẵn code để chạy qua `ngrok` hoặc Streamlit Cloud để khách hàng không cần cài đặt.
2. **Dashboard UI:** 
   - Màu sắc chuyên nghiệp (Xanh Navy / Trắng).
   - Sidebar dễ điều hướng.
   - Hiển thị thông tin tổng quan (KPIs): Tổng số dự án, Tổng số đối tác, Tổng số contacts.
3. **Features:**
   - **Bảng Động:** Có filter theo tỉnh thành, phân khúc.
   - **Nút Export:** Click một phát ra file Excel đẹp.
   - **RAG Chatbot:** Phải có giao diện chat thân thiện, có gợi ý câu hỏi (Suggested Questions).
4. **Transparency:** Cuối mỗi câu trả lời của AI, hãy có một dòng nhỏ: *"Độ chính xác: 100% dựa trên [Tên file nguồn]"*.

## 📈 Các Option Tư vấn (Consulting Menu):
Claude hãy chuẩn bị một script nhỏ hoặc một trang trong Dashboard ghi rõ 3 mức giá/lựa chọn (Lite, Pro, Enterprise) để khách hàng có thể lựa chọn sau khi xem demo.
